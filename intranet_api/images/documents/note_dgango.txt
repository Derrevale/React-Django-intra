Etape 1 installation de python
	Etape 1. Verification de l'installation
		py -V
		
Etape 2 création d'un repository git
	Etape 2.1 Cloner le repository vers la machine local
	
Etape 3 Création d'un Venv 
		py -m venv venv
		
Etape 4 Connection au Venv /!\ n'aime pas les connection a des dossier distant dans un CMD pas Powershell
		venv\Scripts\activate.bat  
		
Etape 5 installation Django
		pip install django
	Etape 5.1 verification
		django.get_version()
		
Etape 6 Créer le projet
		django-admin startproject silva_intranet
	Etape 6.1 rejoindre le dossier du projet 
		cd silva_intranet
	Etape 6.2 mettre a jour en fonction des requirements
		pip install -r requirements.txt
		
Etape 7 Lancer le serveur
		py .\manage.py runserver
		
Etape 8 création d'un superuser
		py manage.py createsuperuser		

Etape 9 création de nouvelle App
		django-admin startapp AppName

Etape intermediaire 
		pip freeze > requirements.txt
		
		py manage.py makemigrations
		
		py manage.py migrate
